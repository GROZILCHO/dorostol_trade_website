# Visual Direction

## Status
This document defines the visual-direction baseline for future refinement. It is a planning reference, not an implementation task. No runtime assets or frontend files are changed by this baseline.

## Corrected Brand Identity
The temporary leaf-style project mark currently used in the frontend is not the official Dorostol Trade logo identity.

The official identity is based on:
- a phoenix / tower mark;
- a dark blue tower structure;
- orange and red fire elements;
- a brick-brown wordmark;
- the optional slogan: "Conditio, sine qua non!"

Future visual work must align the plant-project website with this official identity while preserving the project's industrial and agricultural context.

## Visual Concept
The target concept is an industrial agritech investment platform for a vegetable protein and oil extraction plant.

The site should feel:
- credible;
- industrial;
- investment-grade;
- technical;
- premium but restrained;
- connected to agricultural raw materials.

The site should not feel:
- like a generic SaaS landing page;
- like a green eco brochure;
- like a commodity trading site;
- like an engineering-services site;
- like a fake futuristic factory render;
- like a wellness or organic consumer brand.

## Corrected Color System
### Brand Palette
| Role | Color | Usage |
| --- | --- | --- |
| Primary brand dark blue | `#2A475E` | Main brand anchor, strong backgrounds, header and footer treatments |
| Accent orange | `#E55934` | CTAs and small accent elements |
| Light neutral | `#F0F2F5` or `#F8FAF7` | Light backgrounds and breathing space |
| Neutral text | `#2A2A2A` | Primary body and interface text |
| Brick brown | `#6A3E39` | Wordmark-aligned supporting detail |
| Secondary blue-gray candidate | `#AECD56` | Requires visual review: the provided value and the described muted blue-gray appearance do not appear to match |

### Project Support Palette
| Role | Color | Usage |
| --- | --- | --- |
| Deep green | `#142013` | Sustainability or agriculture-supporting blocks |
| Industrial green | `#4F7043` | Supporting accents and agricultural context |
| Warm stone | `#EEE8DC` | Restrained section backgrounds |
| Muted gold | `#A48955` | Limited supporting detail |

### Color Rules
- Dark blue should become the main brand anchor.
- Orange should be used for CTAs and small accent elements.
- Green should support sustainability and agriculture themes, not dominate the entire site.
- Avoid overusing fire orange or red.
- Avoid a purely green identity that conflicts with the official logo.
- Review the secondary blue-gray candidate before implementation because the documented hex value and visual description require confirmation.

## Logo Usage
- Replace the temporary leaf mark in a future scoped implementation.
- Use official Dorostol Trade logo assets only after they are added to approved runtime assets.
- Use a compact logo treatment suitable for navigation in the header.
- The footer may use a wider or more complete logo treatment.
- Use a mark-only version for favicon and small contexts.
- Do not stretch, recolor, animate, shadow, or decorate the logo.
- Do not place the logo on busy imagery without contrast control.
- Do not use an oversized mascot or logo as decorative filler.
- Do not treat the slogan as mandatory in every layout.

## Image Strategy
### Allowed Image Themes
- soybean, sunflower, and rapeseed macro photography;
- vegetable-oil close-ups;
- plant-protein and protein-isolate powder textures;
- clean food-grade industrial equipment details;
- stainless-steel piping and processing lines;
- industrial laboratory and sample-container visuals;
- approved site photos from Andrasesti;
- approved architectural or industrial plant renders;
- stylized location visuals for Andrasesti, Bucharest, and Constanta;
- process diagrams or abstract technical overlays.

### Avoid
- generic handshakes;
- fake employees;
- unrelated factories;
- AI-generated machinery with impossible geometry;
- visible fake text or logos in images;
- stock farming cliches;
- consumer wellness food imagery;
- animal-protein imagery;
- images implying that the plant is already operational unless verified;
- images without public usage rights.

## Image Status Rules
Every image considered for runtime use should be classified internally as one of:
- actual site;
- approved render;
- reference image;
- illustrative stock;
- AI-generated concept.

No image should imply that the plant is completed or operational unless this is verified.

## Section-Level Asset Recommendations
| Section | Recommended Visual Treatment |
| --- | --- |
| Hero | Approved render, site photo, or abstract industrial / agricultural composite |
| Project Overview | Investment snapshot or identity block |
| Market Opportunity | Raw-material or regional-market visual |
| Location | Stylized Romania map with Andrasesti, Bucharest, and Constanta |
| Products | Product-texture cards for proteins, isolate, and oils |
| Capacity | Technical stat-card system |
| Technology | Food-grade process-equipment detail or approved process diagram |
| Sustainability | By-product and resource-efficiency visual |
| CTA / Contact | Restrained corporate block |

## Phase Plan
### Phase 1A
- Refine brand alignment without adding new runtime imagery.
- Replace the temporary mark with approved official logo assets if available.
- Align colors, spacing, typography, hero, stat cards, and footer with the official identity.

### Phase 1B
- Integrate images in a controlled way after asset approval.

### Phase 2
- Add real project photos, approved renders, diagrams, a gallery, and possible multi-page expansion.

## Implementation Gate
No visual-refinement implementation should start until:
- official logo assets for runtime use are selected;
- asset filenames and locations are approved;
- this visual direction is reviewed;
- the first visual-refinement task has a strict file scope.
